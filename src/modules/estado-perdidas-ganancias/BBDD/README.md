# BBDD - Estado de Pérdidas y Ganancias

Carpeta reservada para estructuras, datos demo, documentación de tablas y futuras consultas del módulo financiero.

## Datos esperados

- Periodos contables.
- Líneas de ingresos.
- Líneas de costos.
- Gastos operativos.
- Otros ingresos/gastos.
- Impuestos.
- Totales y márgenes calculados.

## Notas

El módulo está disponible actualmente en el `index.html` estático. Cuando se migre a componentes Next, los datos y esquemas deben registrarse aquí.
